# KnightStarRating
一个五星评分
使用方法如下
```
    _starRating.backgroundColor  = [UIColor whiteColor];
    _starRating.normalImage = [UIImage imageNamed:@"img-xing-gray"];
    _starRating.selectImage = [UIImage imageNamed:@"img-xing-blue"];
    _starRating.maxRating = 5.0;
    _starRating.delegate = self;
    _starRating.horizontalMargin = 0;
    _starRating.editable = NO;
    _starRating.rating = 2.5;
    _starRating.displayMode = JKStarRatingDisplayHalf;
```
